# Build static binaries for urunc supported monitors

This repository serves as a reference for building and downloading
[urunc](https://github.com/urunc-dev/urunc) supported monitors.

Currently supported monitors and tools:
- [Qemu](https://www.qemu.org/)
- [Firecracker](https://firecracker-microvm.github.io/)
- [Solo5](https://github.com/Solo5/solo5)
- [Virtiofsd](https://gitlab.com/virtio-fs/virtiofsd)
